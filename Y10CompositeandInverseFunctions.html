Chapter 7: Composite and Inverse Functions

Section 7.1: What are Composite and Inverse Functions?

A composite function is a type of function that is created when another function is substituted into it. Let's say we have f(x) = x + 1 and g (x) = 2x as functions. When we say f(g(x)), it means to insert g(x)'s function into f(x), so f(g(x)) = 2x + 1 as 2x replaces x in f(x). We can also insert g(x) into f(x), we substitute x + 1 to replace x in 2x, so we get g(f(x)) = 2(x + 1).
An inverse function is a type of function that is created when you switch the places of x and y, and if f(x) is our function, its inverse is denoted by f-1(x), any function notation with -1 indicates that it's the inverse of that function. For example we have f(x) = 2x + 1, we can make x the subject of the function while using f(x) as y, and we have x = 1/2(y - 1), and f-1(x) = 1/2(x - 1) after we switched the places of x and y.





Section 7.2: Solving Composite Functions

Composite function - related questions sometimes include symbols such as (f o g)(x), which means the same as f(g(x)), but is denoted simpler. Sometimes it involves the symbols (f o g o h)(x) or something similar, it means f(g(h(x))), but the previous form does look neater doesn't it? If there are more than 2 functions however, you have to follow the order in substituting the functions. In (f o g o h)(x), you have to work from the right to the left, meaning substituting h(x) into g(x) first, and its result then substituted into f(x).

Section 7.3: Solving Inverse Functions

Sometimes questions ask you to find the inverse of a function, let's say f(x). We know how to do that from Section 7.1. If we needed to find f-1(3), you would think the first step is to find f(x)'s inverse and substitute x = 3 into it right? What if I told you there's a simpler way to find it? There's a rule for inverse functions that can help you skip that step. f-1(3) has the same value with f(x) = 3. Make the function equal to the x value will help you find the result of when the x value is substituted into the function's inverse. This speeds up your process of thinking and helps you out when not much information is given.








Section 7.4: Rules Relating Composite and Inverse Functions

a. (f-1 o f)(x) = (f o f-1)(x) = x

When you input a function into its inverse or vice versa, they cancel out and the x value remains.

b. (f o f-1 o g)(x) = g(x)

Still related to the previous rule, having a function and its inverse as the first two functions stated will cancel each other out, leaving the function closest to '(x)' alone. However, since the order matters, (f o g o f-1)(x) will not be equal to g(x).



c. (g-1 o f-1)(a) --> (g-1 o f)(x) = a --> f(x) = g (a)

As mentioned in Section 7.3, you can 'switch sides' with the functions by finding their inverses. This applies for this rule. You can move a to the other side, as well as g(x), so it makes g(a).

d. (f o g)-1 = (g-1 o f-1)(x)

When the inverse sign (-1) is 'collective' or outside a set of parentheses with more than one function, it can be distributed inside. It flips the order of the functions and inverses them both.
